# Agent Rules

- Main brand color: `#2B985E`.

## Running

- Install deps: `npm install`
- Expo dev server: `npm start`
- Expo Android: `npm run android`
- Web preview: `npm run web`
- Typecheck: `npm run typecheck`
- Build Android APK: `npm run android:apk`
- Install/run on Waydroid: `npm run android:waydroid`
- Install/run on Genymotion: open a Genymotion device, then `npm run android:genymotion`
